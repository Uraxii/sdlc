# SDLC: Docs-only

Docs, comments, or README changes only.

## Sequence
```
Skeptic
```

## Steps

1. **Skeptic**: verify accuracy against current code.
   - Claims match actual implementation?
   - No outdated API references?
   - No instructions that break if followed?
   Verdict: Approved / Revise / Rejected.

> No relay, no version bump, no Friction Reviewer required.
